# msg-dashboard-back-end

API em .NET (C#) para leitura e extração automática de e-mails `.msg`.

## Como rodar
1. Coloque seus arquivos `.msg` na pasta `Emails`.
2. Rode o comando: `dotnet run --launch-profile "http"`

---
⚠️ **Importante**: Este repositório é apenas a API. Para visualizar a interface gráfica e interagir com os e-mails, você precisará rodar em conjunto o [msg-dashboard-front-end](https://github.com/Lucas-Github-23/msg-dashboard-front-end).
